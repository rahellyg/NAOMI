  2. מהמחשב: UPDATE_JSON_KEY (או GITHUB_TOKEN) + node tools/push-availability.mjs rahellyg NAOMI – ראי github-schedule-sync.md.
  3. git add → commit → push אם צריך; המתיני ל-Pages ורענון קשיח (Ctrl+F5).
מדריך מפורט: github-schedule-sync.md בתיקיית הפרויקט.

יומן זמינות

כל הימים פתוחים כברירת מחדל. לחיצה על תאריך עתידי = סוגרת אותו לקביעות (אדום). לחיצה שוב = פותחת. השעות והמרווח בכרטיס „שעות” למטה.

חודשים שעברו באפור. בסוף: שמור + עדכון הקובץ באתר (ראי התיבה הצהובה למעלה).
