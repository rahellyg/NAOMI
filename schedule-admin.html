ניהול זמינות לפגישות

דף פרטי – אל תשתפי קישור. שמור דוחף ל-GitHub עם טוקן GitHub (PAT) – הדביקי בשדה למטה או השתמשי ב־localStorage / window.UPDATE_JSON_KEY (ראי תיבה צהובה).

כניסה

סנכרון: index.html טוען naomi-availability.json; אם fetch נכשל – גיבוי naomi-availability-embed (שמרו זהה לקובץ). טוקן: Secrets ב-GitHub לא נגישים לדפדפן – הדביקי PAT בשדה למטה או localStorage/window.UPDATE_JSON_KEY (לא ב-commit). שמור דוחף לריפו; אם CORS – העתק ללוח או node tools/push-availability.mjs. פירוט: github-schedule-sync.md.

יומן זמינות

כל הימים פתוחים כברירת מחדל. לחיצה על תאריך עתידי = סוגרת אותו לקביעות (אדום). לחיצה שוב = פותחת. השעות והמרווח בכרטיס „שעות” למטה.

פתוח לקביעה חסום יש פגישה

חודשים שעברו באפור. מסגרת זהובה = יום עם פגישה מהרשימה למטה. בסוף: שמור (טוקן מהשדה למטה, או localStorage / window).

פגישות שנקבעו

דפדפן אחר / טלפון: רשימת הפגישות מגיעה מ־GitHub (קובץ naomi-bookings.json בענף main) – לא מ־localStorage של הלקוחות. אחרי כל הזמנה מהדף הציבורי חייב להתעדכן הקובץ ב־repo: BOOKING_LEDGER_WEBHOOK (Make → commit) או עדכון ידני + שמור פגישות ל-GitHub. רענון מושך מחדש מ־GitHub (זהה בכל מכשיר). localStorage רק במחשב שבו נפתח דף ההזמנה – לא מחליף את הקובץ בשרת. וואטסאפ ללקוחה: כפתור ירוק בכל פגישה.

    שעות

    שעת התחלה ושעת סיום כוללות דקות (למשל 09:30–18:45). סוף החלון = המועד האחרון שבו פגישה יכולה להסתיים.

    טוקן GitHub (PAT)

    להדבקה לפני שמור — אותו הרשאות כמו ל־UPDATE_JSON_KEY (repo, תוכן). הטוקן לא נשלח לשרת של האתר; רק ל־API של GitHub מהדפדפן שלך.
