# חיבור הטופס ל-Google Sheets

## שלב 1: יצירת הגיליון
1. היכנסי ל-[Google Sheets](https://sheets.google.com) וצרי גיליון חדש.
2. בשורה הראשונה כתבי כותרות: **תאריך** | **שם** | **אימייל** | **טלפון** | **הודעה**

## שלב 2: הוספת הסקריפט
1. בתפריט: **הרחבות** (Extensions) → **Apps Script**.
2. מחקי את הקוד שמופיע והדביקי את הקוד מהקובץ `GoogleAppsScript.gs`.
3. שמרי (Ctrl+S) ונתני שם לפרויקט (למשל "טופס נחיתה").

## שלב 3: פרסום כאפליקציית ווב
1. לחצי **פרוס** (Deploy) → **פריסה חדשה** (New deployment).
2. ליד **סוג** (Type) לחצי על האייקון ובחרי **אפליקציית אינטרנט** (Web app).
3. הגדרות:
   - **תיאור**: אופציונלי
   - **הרץ בתור**: אני (חשבון המייל שלך)
   - **גישה**: כל אחד (Anyone) – כדי שהטופס מהאתר יוכל לשלוח נתונים
4. לחצי **פרוס** (Deploy), אשרי הרשאות אם מתבקש.
5. **העתיקי את כתובת האפליקציית האינטרנט** (Web app URL) – זה ה-URL שמתחיל ב-`https://script.google.com/...`

## שלב 4: הדבקת הכתובת בדף
1. פתחי את הקובץ `index.html`.
2. חפשי את השורה: `var GOOGLE_SCRIPT_URL = '';`
3. בין המרכאות הדביקי את כתובת אפליקציית האינטרנט ששמרת. לדוגמה:
   ```javascript
   var GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/AKfycbz.../exec';
   ```
4. שמרי את הקובץ.

מעכשיו כל שליחה מהטופס תישמר אוטומטית בגיליון, ובמקביל ייפתח וואטסאפ עם ההודעה המוכנה.
